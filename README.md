# Task Management Dashboard (React)

This is a simple task management dashboard built with React to practice and demonstrate core frontend development skills.

The application allows users to manage daily tasks through a clean and intuitive interface, focusing on usability, state management, and real-world frontend logic.

## Features
- Create, complete, and delete tasks
- Filter tasks by status (All, Completed, Pending)
- Save tasks automatically using browser localStorage
- Clean, responsive, and easy-to-use UI

## Tech Stack
- React
- JavaScript
- HTML
- CSS

## Why this project
I built this project to strengthen my understanding of React components, state handling, and user interaction while working on a realistic frontend use case. It reflects how I approach building practical, maintainable web interfaces.
